Task 3.2.2.a.

Find the relation between the tension \(T\) and weight \(W\) for the system of FigureÂ 3.1 using unit vectors.

Answer
Solution

Express the forces in terms of their magnitudes and the unit vector \(\jhat\) then proceed as before,

\begin{align*} \Sigma \vec{F}= 0\\ \vec{T}+ \vec{W} \amp= 0\\ T\jhat + W (-\jhat ) \amp = 0\\ T \cancel{\jhat} \amp = W \cancel{\jhat}\\ T \amp = W \end{align*}
