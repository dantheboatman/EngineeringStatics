In this section we will use the integration process describe above to calculate the area of the general spandrel shown in FigureÂ 7.7.3. A spandrel is the area between a curve and a rectangular frame. This is a general spandrel because the curve is defined by the function \(y = k x^n\text{,}\) where \(n\) is not specified. If \(n = 0\) this is a constant function, if \(n=1\) this is a straight line, \(n=2\) is a parabola, etc.. You can change the slider to see the effect of different values of \(n\text{.}\)
