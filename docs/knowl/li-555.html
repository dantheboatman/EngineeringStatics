In the approach, the load is described as a piecewise function of \(x\text{,}\) which is integrated twice to develop equations for shear and moment. This method can used for arbitrarily complicated loading distributions, and can be used by software solutions. Applying this method by hand requires accurate integration and differentiation, and application of boundary conditions.
