A third way to represent a vector is with its unit vector multiplied by a scalar value called its scalar component. A unit vector is a vector with a length of one (unitless) which points in a defined direction. Hence, a unit vector represents pure direction, independent of the magnitude and unit of measurement. The scalar component is a signed number with units that may be positive or negative, and which defines both the magnitude and sense of the vector. They should not be confused with vector magnitudes, which are always positive.
