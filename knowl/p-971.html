For example, if the resultant force is only in the \(x\) direction and the resultant moment acts only around the \(x\) axis, this is an example of a wrench resultant. An everyday example is a screwdriver, where both the resultant force and axis of rotation are in-line with the screwdriver. A wrench resultant is considered positive if the couple vector and force vector point in the same direction, and negative if they point in opposite directions.
