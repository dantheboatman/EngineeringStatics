When all forces in a system are coplanar we need only find the resultant force in this plane and the resultant moment about the axis perpendicular to this plane. That is, if all forces exist in the \(x\)-\(y\) plane, we need only to sum components in the \(x\) and \(y\) directions to find resultant force \(\vec{R}\text{,}\) and use these to determine the resultant moment about the \(z\) axis. All two-dimensional problems fall into this category.
