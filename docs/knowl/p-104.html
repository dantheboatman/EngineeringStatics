The standard notation for a vector uses either an arrow or bar above the vector’s name or the vector’s name in bold font. All three of these notations mean the same thing.
\begin{equation*} \vecarrow{F} = \bar{F} = \vec{F} = \textrm{ a vector named } F \end{equation*}
Most printed works including this book will use the bold symbol for vectors, but for handwritten work, you and your instructor will use the bar or arrow notation.
