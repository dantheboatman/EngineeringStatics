The centroid of rectangle 2 is located \(\mm{70}\) above the \(x\) axis so we must use the parallel axis theorem (10.3.1), so
\begin{align*} (I_x)_2 \amp = \bar{I} + A d^2\\ \amp = \frac{b h^3}{12} + (b h) d^2 \\ \amp = \frac{(90)(20)^3}{12} + (90 \times 20) ( 70)^2 \\ (I_x)_2 \amp = \mm{8.88 \times 10^6}^4\text{.} \end{align*}
